Graph Viewer Project
===============================


On Linux
-------

Dependencies
---------------

sudo apt-get install python3-pyqt5

sudo pip3 install pyqtgraph

sudo pip3 install numpy

Run
---------------

python3 graphviewer.py



On Windows (ran on Cygwin)
-------

Dependencies
---------------

pip install PyQt5

pip install pyqtgraph

pip install numpy

Run
---------------

python graphviewer.py
